
# Jazz Festival Web Application

## Project Overview

This project is a frontend web application for a Jazz Festival website, developed as a faculty project.
The application is built using Vue.js and represents the client-side of a full-stack system.

## Technologies Used

- Vue.js 3
- Vite
- Vue Router
- HTML5
- CSS3
- JavaScript (ES6+)

## Project Setup

Install dependencies
npm install

Start development server
npm run dev

## Compile and Minify for Production

npm run build
