
# Skopje Jazz Festival - Web Application

## Project Overview

This project represents a frontend web application developed for the Skopje Jazz Festival. The application serves as the official digital platform for presenting festival information, including upcoming events, featured artists, and essential festival details.
The goal of this project is to provide users with an intuitive and visually appealing interface where they can explore festival events, view artist profiles, and stay informed about performances and schedules.
The application is built using modern frontend technologies and follows component-based architecture principles using Vue.js 3.

## About the Festival

The Skopje Jazz Festival is one of the most recognized and respected music festivals in the region. It brings together internationally acclaimed jazz musicians and talented artists, offering a rich and diverse musical experience.

This web application allows visitors to:

- Browse festival events by date
- View detailed information about each event
- Explore artist profiles
- See event images and performance details
- Navigate easily through the festival schedule

## Technologies Used

The project is developed using the following technologies:

- Vue.js 3 – Progressive JavaScript framework for building user interfaces
- Vite – Fast frontend build tool and development server
- Vue Router – For managing navigation between pages
- HTML5 – Markup structure
- CSS3 – Styling and responsive design
- JavaScript (ES6+) – Application logic

## Application Structure

The application is structured using Vue components to ensure modularity and maintainability. Key sections include:

- Home Page – Introduction and festival highlights
- Events Section – Calendar and detailed view of festival events
- Artists Section – Artist information and performance details
- Navigation System – Smooth routing between pages using Vue Router

## Project Setup

### Install dependencies

npm install

### Start development server

npm run dev

The application will run locally on the development server.

### Compile and Minify for Production

npm run build

This command generates an optimized production-ready version of the application.
